# 快速开始指南

## 📝 三步开始写作业

### 1. 复制模板
```bash
cp homework_template.typ 我的作业.typ
```

### 2. 修改标题（打开 `我的作业.typ` 找到标题页部分）
```typst
#align(center)[
  #text(size: 20pt, weight: "bold")[拓扑学 作业 1]  // ⬅️ 改这里
  
  #v(1em)
  
  #text(size: 14pt)[
    姓名：张三 \          // ⬅️ 改这里
    学号：20233000000     // ⬅️ 改这里
  ]
  
  #v(0.5em)
  
  #text(size: 12pt)[
    提交日期：2025年11月4日  // ⬅️ 改这里
  ]
]
```

### 3. 编译生成 PDF
```bash
typst compile 我的作业.typ
```

## 🎯 课程名称示例

只需修改标题中的"课程名称"即可用于任何数学课程：

| 课程 | 标题示例 |
|------|---------|
| 拓扑学 | `[拓扑学 作业 1]` |
| 微分几何 | `[微分几何 作业 2]` |
| 实分析 | `[实分析 作业 3]` |
| 泛函分析 | `[泛函分析 作业 4]` |
| 抽象代数 | `[抽象代数 作业 5]` |

## 📚 编写作业内容

删除模板中的示例问题，添加你自己的问题：

```typst
#problem("1")[
  这里写第一题的题目
]

#solution[
  这里写第一题的解答
  
  // 可以使用定理环境
  #theorem(title: "定理 1.1")[
    ...
  ]
  
  // 可以使用证明环境
  #proof[
    ...
  ]
]
```

## 🔧 常用符号（拓扑学）

| 输入 | 输出 | 说明 |
|------|------|------|
| `$interior(A)$` | Å | 内部 |
| `$closure(A)$` | A̅ | 闭包 |
| `$boundary(A)$` | ∂A | 边界 |
| `$A inter B$` | A ∩ B | 交集 |
| `$A union B$` | A ∪ B | 并集 |

## 💻 添加代码块

```typst
#code_block(
  ```python
  import numpy as np
  x = np.array([1, 2, 3])
  print(x)
  ```,
  caption: "代码 1: Python 示例"
)
```

支持 Python, MATLAB, R, C++, Java 等多种语言。

## 📎 添加附录

```typst
#appendix_title[附录]

#appendix_section[A. 代码][
  #code_block(
    ```python
    # 你的代码
    ```,
    caption: "代码说明"
  )
]
```

附录会自动另起一页，适合放置代码、补充材料等。

详细说明请查看 `README.md`。
